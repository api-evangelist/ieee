# IEEE Xplore

IEEE Xplore is the authoritative technical library operated by the Institute of Electrical and Electronics Engineers (IEEE), providing access to over 6 million documents spanning engineering, computer science, electronics, and technology.

The IEEE Xplore API suite exposes a REST-based Metadata Search API for querying and retrieving abstracts and metadata across the full corpus, a DOI API for resolving up to 25 DOIs per request, an Open Access API for full-text retrieval of freely available articles, and a Full-Text Access API for institutional subscribers.

## APIs

- **Metadata Search API** - Query and retrieve metadata and abstracts for 6M+ IEEE documents
- **DOI API** - Batch lookup of up to 25 DOIs per request
- **Open Access API** - Full-text retrieval for open-access articles
- **Full-Text Access API** - Full-text retrieval for institutional subscribers

## Base URL

`https://ieeexploreapi.ieee.org/api/v1/search/articles`
